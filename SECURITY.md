# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to don@altlaboratories.com

## Security Measures

### Authentication
- JWT tokens with expiration
- Token rotation
- Secure password hashing (bcrypt)

### Authorization
- Role-based access control
- Row-level security (Supabase RLS)

### Data Protection
- HTTPS everywhere
- Input validation
- SQL injection prevention
- XSS protection
- CSRF tokens

### API Security
- Rate limiting
- Request validation
- Error message sanitization
- Security headers

### Infrastructure
- Regular security updates
- Dependency scanning
- Secret rotation
- Backup encryption

## Compliance

- GDPR compliant
- Data encryption at rest
- Right to deletion support
